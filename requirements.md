# ENERGY CATCHER 랜딩 페이지 요구사항 명세서

## 1. 개요
- **회사명**: DainSynergy
- **서비스명**: ENERGY CATCHER
- **목적**: 산업용 장비 모니터링 서비스를 제공하는 회사의 첫인상을 결정짓는 랜딩 페이지 구축
- **핵심 목표**: 모던하고 심플한 디자인을 통해 서비스의 신뢰성을 어필하고, 사용자로 하여금 앱(App Store) 및 데스크탑 버전 다운로드를 유도

## 2. 주요 구성요소 및 섹션
1. **헤더 (Header)**: 
   - 회사명(DainSynergy) 및 서비스 로고 (ENERGY CATCHER)
   - 주요 네비게이션 액션
2. **히어로 섹션 (Hero Section)**:
   - 강렬하고 직관적인 메인 카피
   - 서비스 화면 Mockup 이미지 또는 그래픽
   - 핵심 CTA (Call To Action) 버튼: **App Store 다운로드** 및 **데스크탑 버전 다운로드**
3. **서비스 소개 (Features)**:
   - 실시간 모니터링, 알림(Alert), 데이터 분석 등 주요 기능 설명
4. **다운로드 (Downloads)**:
   - 다운로드 링크 명시적 제공 영역
5. **푸터 (Footer)**:
   - DainSynergy 회사 정보, 이메일, 저작권 등

## 3. 디자인 가이드
- **메인 컬러**: `#F45A5A` (핑크 계열). 강렬한 색상이므로 전체 배경에 남용하기보다, 사용자의 시선을 끌어야 하는 핵심 CTA 버튼, 강조 텍스트, 아이콘 등의 포인트 컬러로 신중하게 배치하여 시각적 피로도를 낮추고 모던함을 극대화함.
- **배경 및 보조 컬러**: 화이트 및 라이트 그레이(Light Gray) 기반으로 구성하여 산업용 솔루션의 깔끔함과 신뢰감을 강조.
- **타이포그래피**: `Inter` 폰트 전면 사용. 회사명, 서비스명(ENERGY CATCHER), 섹션 타이틀 등에는 **Bold** 웨이트를 사용하여 뚜렷하고 강렬한 인상을 줌.
- **스타일**: '모던 & 심플' 스타일 유지.
- **반응형 웹 (Responsive Web)**: 모바일, 태블릿, 데스크탑 등 지원.

## 4. 기술 스택
- HTML5, CSS3 (바닐라 CSS 활용)
- Javascript (UI 인터랙션용)
